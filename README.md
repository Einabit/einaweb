### EinaWeb

It's just a tool for creating static sites with steroids

### Install

`npm i -g einaweb`

### Samples

https://github.com/Einabit/web

More details comming soon

```
$ einaweb help
Available subcommands

  - init          initializes directory structure
  - build         generates the whole web
  - watch         same as build but with a preview server
  - version       prints the script's version
  - help          prints this message
```

### SVC tracking

ignore these files: public/*.html as you're just generating all HTML files so you don't need the SVC to track them
